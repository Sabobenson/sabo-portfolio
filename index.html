Small Office Network Routing Setup

By Sabo Benson – Telecommunication & Network Engineer

Problem

Devices in two different networks were unable to communicate due to lack of routing. This limited data sharing between departments in a small office environment.

Solution

A router was configured to enable communication between the two networks using proper IP addressing and interface configuration.

  • Designed network topology
  • Configured router interfaces
  • Assigned IP addresses
  • Tested connectivity

Network Topology

PC1 → Switch1 → Router → Switch2 → PC2

(Insert your topology image here)

IP Addressing

  • PC1: 192.168.1.2 /24
  • Router G0/0: 192.168.1.1
  • Router G0/1: 192.168.2.1
  • PC2: 192.168.2.2 /24

Configuration

enable configure terminal interface g0/0 ip address 192.168.1.1 255.255.255.0 no shutdown interface g0/1 ip address 192.168.2.1 255.255.255.0 no shutdown end

Testing

Ping test from PC1 to PC2:

ping 192.168.2.2

Result: Successful communication with no packet loss.
